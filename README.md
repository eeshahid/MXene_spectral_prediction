# Optimizing Spectral Prediction in MXene-Based Metasurfaces Through Multi-Channel Spectral Refinement and Savitzky-Golay Smoothing

<img width="1600" height="1066" alt="Graphical_abstract" src="https://github.com/user-attachments/assets/9ca855db-f212-43e1-a334-19f8a39399cf" />

Deep learning framework for rapid spectral prediction of MXene-based metasurface absorbers using transfer learning, Multi-Channel Spectral Refinement (MCSR), and Savitzky–Golay smoothing.

The proposed approach enables fast and accurate surrogate modeling for nanophotonic design, significantly reducing the computational cost of full-wave electromagnetic simulations for applications in solar harvesting, sensing, and spectral engineering.
